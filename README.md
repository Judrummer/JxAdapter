# JxAdapter
Simple and Powerful RecyclerAdapter for Android Kotlin


## Installation

### Gradle

``` Groovy
repositories {
   maven { url "https://jitpack.io" }
}

dependencies {
    compile 'com.github.judrummer.jxadapter:jxadapter:0.1.0' 
    compile 'com.github.judrummer.jxadapter:jxadapter-rxjava:0.1.0' 
}
```
